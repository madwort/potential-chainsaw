# Potential Chainsaw

Reimplement jacktrip in Rust

By Tom Ward / Madwort

https://github.com/madwort/potential-chainsaw

## Install & run

```bash
$ cargo build
$ ./target/debug/jacktrip-rust
```

Test/demo app:

```bash
$ cargo build --bin udpserver
$ ./target/debug/udpserver
```

## Dependencies 

[rust-jack](https://github.com/RustAudio/rust-jack)
